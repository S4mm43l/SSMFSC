# SSMFSC

## Setup Project

### Backend

```bash
cd backend
npm install
```

### Frontend

```bash
cd frontend
npm install
```

## Launch project

### Backend

```bash
cd backend
npm run start:dev
```
#### ** backend will start at http://localhost:3000 by default**

### Frontend

```bash
cd frontend
npm run dev
```
#### ** frontend will start at http://localhost:5173 by default**
